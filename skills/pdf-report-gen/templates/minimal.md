# Template MINIMAL — Brief

## Audience cible

Lecteurs qui veulent du **contenu pur sans fioritures**. Documents destines a impression N&B, archivage long terme, ou contextes austeres (juridique, regulatoire interne).

## Style visuel

- **Police** : Helvetica / Arial sans-serif
- **Couleurs** : noir et blanc UNIQUEMENT (zero couleur)
- **Mise en page** : monobloc, lineaire, sans encadres
- **Footer** : minimaliste (juste pagination)
- **Pas de** : KPI cards, callouts colores, gradients, effets

## Quand l'utiliser

| Situation | Verdict |
|-----------|---------|
| Document juridique / regulatoire | OUI |
| Archive long terme (impression N&B) | OUI |
| Note interne austere | OUI |
| Brouillon / draft a iterer | OUI |
| Demande utilisateur "le plus simple possible" | OUI |
| Pitch board | NON -> `executive` |
| Analyse boursiere | NON -> `financial` |

## Structure type

```
Page de titre simple (titre + auteur + date + classification)
Table des matieres (texte simple, pas d'icones)
1. Section 1
2. Section 2
3. ...
Conclusion
Annexes (texte)
```

## KPIs

**Pas de KPI cards** dans ce template. Si necessaire, integrer les chiffres dans le texte ou en tableau classique.

## Visualisations

- **Graphiques** : OK mais en niveaux de gris
- **Mermaid** : OK avec theme `default` neutre
- **Tableaux** : oui, sans couleur de fond
- **Pas d'images decoratives**

## Anti-patterns

- Ajouter des couleurs "pour egayer" -> contredit le brief
- KPI cards -> n'existent pas dans ce template
- Callouts colores -> remplacer par texte en italique ou note de bas de page
- Logos colores -> convertir en N&B ou supprimer

## Exemple de cas d'usage

> "Note interne archivable a long terme sur la procedure d'audit interne 2026, format austere, impression N&B"
