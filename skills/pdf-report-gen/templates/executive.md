# Template EXECUTIVE — Brief

## Audience cible

Board, CEO, comite de direction, decision-makers strategiques. Lecteurs presses qui veulent **conclusions et decisions** en 5 minutes.

## Style visuel

- **Police** : Georgia serif (style "consulting top-tier")
- **Mise en page** : aeree, beaucoup de blanc, peu de densite
- **Titres** : grands, capitales pour H1
- **Couleurs** : palette sobre (noir/gris/bleu marine accent)
- **Footer** : minimaliste, pagination discrete

## Quand l'utiliser

| Situation | Verdict |
|-----------|---------|
| Note strategique pour board | OUI |
| Synthese trimestrielle CEO | OUI |
| Recommandation acquisition | OUI |
| Memo d'investissement haut de gamme | OUI |
| Analyse technique detaillee | NON -> `technical` |
| Analyse boursiere dense | NON -> `financial` |
| Etude academique | NON -> `research` |

## Structure type

```
Cover page (logo + classification + version)
1. Resume executif (TL;DR en 5 phrases max)
2. Decision recommandee (1 phrase actionnable)
3. Contexte strategique (3-4 paragraphes)
4. Analyse SWOT (4 quadrants)
5. Scenarios (Bull / Base / Bear)
6. Risques majeurs (top 3)
7. Next steps (3-5 actions)
Annexes (sources)
```

## KPIs typiques

3 a 5 KPIs **strategiques** (pas operationnels) :
- "Marche adressable", "Avantage concurrentiel", "Risque execution"
- Pas de chiffres comptables detailles -> renvoyer en annexes

## Visualisations recommandees

- 1 graphique synthese (comparaison ou evolution)
- 1 diagramme strategique (Mermaid `graph LR` workflow / `stateDiagram` decision)
- **JAMAIS** plus de 3 graphiques (executive = synthese, pas data dump)

## Anti-patterns

- Tableaux denses 10 colonnes -> illisible pour ce profil
- Jargon technique non explique
- Plus de 10 pages -> contradicts "executive brief"
- Bibliographie style APA -> trop academique

## Exemple de cas d'usage

> "Note board sur opportunite acquisition Acme Corp — recommandation finale + risques + financement"
