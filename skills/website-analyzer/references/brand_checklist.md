# Checklist Analyse de Marque

## 1. Identité visuelle — Éléments à vérifier

### Logo
- [ ] Logo présent et visible sur toutes les pages
- [ ] Logo cliquable → retour accueil
- [ ] Logo haute résolution (pas pixelisé)
- [ ] Favicon cohérent avec le logo
- [ ] Logo adapté en version mobile (simplifié si nécessaire)
- [ ] Fond du logo compatible avec le fond du site

### Palette de couleurs
- [ ] Couleur primaire identifiable (la couleur "signature")
- [ ] Couleur secondaire complémentaire
- [ ] Couleur d'accent pour les CTAs
- [ ] Couleurs neutres pour le texte et les fonds
- [ ] Palette cohérente sur TOUTES les pages
- [ ] Pas de couleurs "orphelines" (utilisées une seule fois)
- [ ] Contraste suffisant texte/fond (WCAG AA)

### Typographie
- [ ] Police principale identifiable
- [ ] Police secondaire (si utilisée) complémentaire
- [ ] Hiérarchie typographique claire (titres, sous-titres, corps)
- [ ] Pas plus de 3 familles de polices
- [ ] Tailles cohérentes entre les pages
- [ ] Graisse (bold, regular, light) utilisée de manière cohérente

### Imagerie et visuels
- [ ] Style photographique cohérent (filtre, ambiance, cadrage)
- [ ] Illustrations dans un style uniforme (si utilisées)
- [ ] Icônes dans un style cohérent (outline OU filled, pas les deux)
- [ ] Qualité d'image homogène (pas de mélange HD/basse résolution)
- [ ] Images pertinentes par rapport au message

---

## 2. Positionnement & Messaging — Questions clés

### Proposition de valeur
- [ ] La promesse est-elle identifiable en < 5 secondes sur la homepage ?
- [ ] Le "quoi" est clair (ce que fait l'entreprise/le produit)
- [ ] Le "pour qui" est clair (cible identifiable)
- [ ] Le "pourquoi" est clair (bénéfice client, pas feature)
- [ ] Différenciation vs concurrents visible

### Tone of voice
- [ ] Registre cohérent (formel/informel, tutoiement/vouvoiement)
- [ ] Vocabulaire adapté à la cible
- [ ] Personnalité de marque perceptible (innovant, sérieux, fun, premium, accessible...)
- [ ] Pas de jargon corporate vide ("solutions innovantes", "leader", "synergie")
- [ ] Storytelling présent (histoire de la marque, fondateurs, mission)

### Preuve et crédibilité
- [ ] Témoignages clients (avec noms/photos si possible)
- [ ] Études de cas / réalisations
- [ ] Chiffres clés (clients, CA, années d'expérience)
- [ ] Logos partenaires / clients
- [ ] Certifications / labels / récompenses
- [ ] Presse / médias

---

## 3. Cohérence cross-pages — Points de vérification

### Éléments persistants
- [ ] Header identique sur toutes les pages
- [ ] Footer identique sur toutes les pages
- [ ] Même style de boutons partout
- [ ] Même style de liens partout
- [ ] Même espacement / marge entre les sections

### Transitions entre pages
- [ ] Pas de rupture visuelle en passant d'une page à l'autre
- [ ] Pages "profondeur 2" (sous-pages) aussi soignées que la homepage
- [ ] Pages légales (CGV, mentions) avec le même template
- [ ] Blog (si existant) visuellement intégré au site principal
- [ ] Espace client / dashboard cohérent avec le site public

### Pages spéciales
- [ ] Page 404 : personnalisée, avec navigation
- [ ] Page de maintenance : brandée
- [ ] Pages de résultats de recherche : stylées
- [ ] Emails transactionnels : cohérents (si vérifiables)

---

## 4. Présence digitale étendue

### Réseaux sociaux
- [ ] Liens vers réseaux sociaux présents
- [ ] Icônes à jour (logos actuels des plateformes)
- [ ] Profils sociaux cohérents visuellement avec le site
- [ ] Contenu actif sur les réseaux (pas de profils abandonnés)

### Partage social
- [ ] Open Graph tags configurés (image, titre, description)
- [ ] Twitter Cards configurées
- [ ] Aperçu de partage soigné (pas de texte tronqué ou image manquante)

### Newsletter / Email
- [ ] Formulaire d'inscription visible
- [ ] Promesse de valeur pour l'inscription (pas juste "inscrivez-vous")
- [ ] Fréquence d'envoi indiquée
- [ ] Désabonnement facile (si vérifiable)

---

## 5. Perception émotionnelle — Grille d'évaluation

Évaluer la première impression en 3 secondes sur ces axes :

| Axe | ←  Négatif | → Positif |
|-----|-----------|-----------|
| Modernité | Daté, vieillot | Moderne, actuel |
| Professionnalisme | Amateur, brouillon | Professionnel, soigné |
| Confiance | Suspect, cheap | Fiable, rassurant |
| Clarté | Confus, surchargé | Clair, aéré |
| Émotion | Froid, impersonnel | Chaleureux, engageant |
| Mémorabilité | Générique, oubliable | Unique, marquant |
| Cohérence | Décousu | Unifié |

Score par axe : 1 (très négatif) à 5 (très positif)
Score perception global : moyenne des 7 axes × 2 = /10
